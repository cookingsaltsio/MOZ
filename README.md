[README.md](https://github.com/user-attachments/files/27289987/README.md)# モンスターおぼえるぜ

「モンスターおぼえるぜ」は、単語や問題をモンスターとのバトル形式で学習するためのブラウザ対応学習アプリです。

PCブラウザ、Android Chrome、iOS Safari、Android APK版で使うことを想定しています。

## できること

- デッキを読み込んで問題演習
- 間隔反復モード（SRS）で復習
- 全項目からの出陣
- 項目一覧からの個別確認
- ミスした問題の復習
- 対戦
- 学習データのエクスポート / インポート
- 音量、ミュート、演出ON/OFFなどの調整

## 対応しているデッキ形式

以下の形式を読み込めます。

- `txt`
- `csv`
- `tsv`
- `xlsx`
- `ods`
- `zip`

`zip` には、複数のデッキファイルや画像ファイルをまとめて入れられます。

`apkg` には現在対応していません。

## デッキ形式の基本

CSV、TSV、Excel、ODSでは、1行を1カードとして読み込みます。

- 1列目: 問題
- 2列目以降: 解答

例:

```csv
question,answer
cat,猫
dog,犬
```

ヘッダー行が `question,answer`、`front,back`、`問題,解答` などの場合は自動でスキップします。

## 学習データの保存について

学習データは、各端末・各ブラウザの中に保存されます。

そのため、次の点に注意してください。

- PCブラウザとスマホブラウザの学習データは自動同期されません。
- Android APK版とブラウザ版の学習データも自動同期されません。
- 端末を変える場合は、エクスポート / インポートでデータを移してください。
- ブラウザのデータ削除やキャッシュ削除をすると、保存データが消える場合があります。

大事な学習データは、定期的にエクスポートしてバックアップしてください。

## 対戦について

対戦は PeerJS / WebRTC を使います。

基本的な流れは次の通りです。

1. 片方が部屋を作ります。
2. 表示された対戦IDを相手に送ります。
3. もう片方が対戦IDを入力して参加します。
4. ホストがデッキや対戦設定を決めます。
5. 問題セットを同期します。
6. 両方が準備OKになったら対戦を開始します。

通信環境やブラウザによっては接続に失敗する場合があります。その場合は、通信ログ内の案内や、TURNリレー設定を確認してください。

## ブラウザで使うときの注意

対戦まで使う場合は、`file://` で直接開くのではなく、HTTPS環境で開いてください。

HTTPSで配信すると、PCブラウザ、Android Chrome、iOS Safariでより安定して動作します。
